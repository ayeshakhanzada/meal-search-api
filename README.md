⬛ Find Meals For Your Ingredients is a place where user searches the ingredients according of its choice and finds the meals recipe with ease. 

⬛ The user can read the instructions of the recipe also it can watch the video of the recipe which is connected to the YouTube links if needed. 

⬛ It uses Free Meal API to show the meals. The website uses Azure services like Azure Blob Storage to host the website.
